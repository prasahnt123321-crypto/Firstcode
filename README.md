# *Purpose of Project:*
# The main objective of this project is to analyze the data of restaurants listed on Zomato.
# Through this we want to understand:
# 1. Which types of restaurants are most popular
# 2. What the distribution of customer ratings looks like  
# 3. What the average cost for two people is
# 4. Extract useful business insights from the data
# *Outputs:*
# Using Python with Pandas, Matplotlib, and Seaborn, 3 main outputs were generated:
# *1. Restaurant Type Analysis*
# - Analyzed data of 148 total restaurants
# - `Dining` type had the highest count with 110 restaurants
# - `Cafes`: 23, `Buffet`: 7, `Other`: 8
# *2. Rating Distribution Analysis*
# - Most restaurants have ratings between 3.5 and 4.25
# - The highest number of restaurants fall in the 3.75-4.25 rating range
# - Very few restaurants have ratings of 4.5+
# *3. Cost Analysis for Two People*
# - Approximate cost for two people ranges from 100 to 950
# - The most common price point is around 300 rupees
# *Benefits:*
# 1. *For Customers*: This analysis helps customers choose the right restaurant based on their budget and preference.
# 2. *For Restaurant Owners*: People opening new restaurants can understand which type has higher demand in the market and what pricing to keep.
# 3. *For Zomato/Business*: Data shows `Dining` is the biggest segment, so more focus can be given to it.
# 4. *Skill Development*: This project gave practical knowledge of data cleaning, data visualization, and Python libraries.
# *Conclusion:*
# The analysis of Zomato data concludes that `Dining` restaurants dominate the market and customers prefer to spend around 300 rupees.
# Most restaurants have above-average ratings of 3.5-4.25, which means service quality is good.
# This project is a good example of how analyzing real-world data can help in making business decisions.
